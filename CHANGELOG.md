# mikumo的咖啡日记 — 更新日志

## V1.0.1 (2026-05-30)

### 新增
- 咖啡豆新增**处理法**字段，提供快捷按钮（日晒 / 水洗 / 蜜处理 / 厌氧 / 半水洗）
- 咖啡豆新增**豆种**字段，提供快捷按钮（阿拉比卡 / 罗布斯塔 / 瑰夏 / 铁皮卡 / 波旁 / 卡杜拉）
- 产地拆分为**产国**、**产区**、**庄园**三个独立字段
- 购买日期改为**烘焙日期**（生产日期），更符合咖啡行业习惯
- 库存卡片新增产地信息和豆种/处理法展示行

### 修复
- 修复应用无图标问题，生成专属咖啡主题图标
- 优化底部导航栏：去掉图标、加大文字（15px 加粗）、紧凑排列
- 修复 Debug APK 在手机上无法独立运行的问题，改用 Release APK

### 构建
- 本地 Gradle 构建 Release APK（77MB，已签名）
